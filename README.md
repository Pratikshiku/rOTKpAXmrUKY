### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Visual Studio Code(VScode)`

- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 家具管理
5. 任务管理


**管理员功能：**
1. 登录
2. 首页
3. 个人中心（修改密码、个人信息）
4. 基础数据管理（家具类型管理、区域管理）
5. 家具管理
6. 任务管理
7. 用户管理
